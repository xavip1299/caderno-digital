# Exercício: Viajantes no Tempo e Cartaz de Evento

## Resumo do Trabalho

Este trabalho gráfico foi desenvolvido ao longo de três fases distintas, cada uma com objetivos e desafios específicos: o **Exercício X ("Viajantes no Tempo")**, o **Exercício Y (Cartaz de Evento)** e a **Adaptação para Instagram**. O projeto demonstra a aplicação prática de técnicas de edição de imagem, design gráfico e adaptação de conteúdo para diferentes plataformas digitais.

---

## Exercício X: "Viajantes no Tempo"

### Conceito
O exercício consistiu em criar uma composição visual que integrasse elementos modernos numa fotografia histórica, criando um efeito de anacronismo deliberado e visualmente coerente.

### Processo de Desenvolvimento

#### 1. Seleção de Elementos
- **Fotografia base**: Escolha de uma fotografia histórica como ponto de partida
- **Elementos anacrónicos**: Seleção cuidadosa de objetos modernos:
  - Uma trotinete elétrica
  - Um cartaz contemporâneo
- Objetivo: criar contraste temporal mantendo coerência visual

#### 2. Edição no Photopea

##### Técnicas de Recorte e Composição:
- **Recorte preciso** dos elementos modernos
- **Posicionamento estratégico** na cena histórica
- **Ajustes de perspetiva** para integração natural
- **Criação de sombras** realistas para dar profundidade

##### Harmonização Visual:
- **Redução de saturação** dos elementos modernos
- **Ajuste de contraste** para corresponder à fotografia antiga
- **Adição de ruído/grão** para simular a textura fotográfica histórica
- **Correção de cor** para harmonizar a paleta cromática

#### 3. Documentação
- Criação de **WIPs (Work in Progress)** para documentar o processo
- Desenvolvimento de **variação do exercício** explorando alternativas
- **Registo das imagens** utilizadas e suas fontes

### Resultados
O exercício resultou numa composição convincente onde elementos de diferentes épocas coexistem de forma visualmente coerente, demonstrando domínio de técnicas de edição e composição digital.

## Exercício Y: Cartaz de Evento - IPMAIA Winter Concert

### Especificações Técnicas
- **Formato**: A3
- **Resolução**: 300 ppi (pixels por polegada)
- **Evento**: IPMAIA Winter Concert
- **Data do evento**: 23 de dezembro de 2025

### Desenvolvimento do Cartaz

#### 1. Estrutura e Layout
- **Montagem no Photopea** com dimensões profissionais
- **Definição de margens** adequadas para impressão
- **Estrutura visual** clara e hierarquizada
- **Composição equilibrada** dos elementos gráficos

#### 2. Hierarquia Tipográfica
Organização visual das informações seguindo princípios de design:
- **Título principal**: Destaque máximo para "IPMAIA Winter Concert"
- **Data**: Informação prioritária visualmente destacada
- **Local**: Posicionamento estratégico
- **Slogan**: Elemento complementar
- **Informações secundárias**: Legibilidade mantida sem competir com elementos principais

#### 3. Identidade Visual

##### Paleta de Cores:
- **Cor institucional**: Laranja do IPMAIA como cor dominante
- **Cores complementares**: Escolhidas para criar contraste e harmonia
- **Consistência cromática**: Alinhamento com a identidade da instituição

##### Elementos Obrigatórios:
- ✓ Título do evento
- ✓ Data (23 de dezembro de 2025)
- ✓ Local
- ✓ Slogan promocional
- ✓ Logótipo do IPMAIA

#### 4. Composição Visual

##### Imagem de Fundo:
- **Seleção criteriosa** de fotografia temática
- **Criação de contraste** através de overlays semitransparentes
- **Garantia de legibilidade** do texto sobre a imagem

##### Design e Alinhamento:
- **Alinhamentos precisos** seguindo grelhas de composição
- **Utilização de fontes adequadas** ao contexto do evento
- **Espaçamento consistente** entre elementos
- **Equilíbrio visual** entre texto e imagem

#### 5. Variações
- Desenvolvimento de **versão alternativa** do cartaz
- Exploração de diferentes abordagens visuais
- Teste de layouts e composições

## Adaptação para Instagram

### Objetivo
Adaptar o cartaz A3 para formatos específicos das redes sociais, mantendo a eficácia comunicativa e identidade visual.

### Formatos Desenvolvidos

#### 1. Instagram Story
**Especificações técnicas:**
- **Dimensões**: 1080×1920 pixels (formato vertical)
- **Aspeto**: 9:16 (tela cheia em dispositivos móveis)

**Adaptações realizadas:**
- **Reorganização do layout** para formato vertical
- **Aumento do tamanho** do título para compensar visualização em ecrã pequeno
- **Ajuste de margens** considerando área de segurança do Instagram
- **Redistribuição de elementos** mantendo hierarquia visual

#### 2. Instagram Post
**Especificações técnicas:**
- **Dimensões**: 1080×1350 pixels (formato 4:5)
- **Aspeto**: Otimizado para feed do Instagram

**Adaptações realizadas:**
- **Recomposição visual** para formato retrato
- **Ampliação de informações essenciais** para legibilidade móvel
- **Ajuste de espaçamentos** adequados ao formato
- **Preservação do logótipo** com dimensão apropriada

### Considerações de Design para Redes Sociais

#### Legibilidade em Ecrãs Pequenos:
- Textos ampliados significativamente
- Contraste reforçado para visualização em diferentes condições de luz
- Hierarquia visual simplificada mas eficaz

#### Exportação:
- **Formato**: PNG (mantém qualidade e transparências)
- **Otimização**: Para visualização digital
- **Conformidade**: Com requisitos do enunciado

## Princípios de Design Aplicados

### 1. Hierarquia Visual
- Organização clara da informação por ordem de importância
- Uso estratégico de tamanho, cor e posicionamento
- Guia natural do olhar do observador

### 2. Contraste
- Contraste cromático para destacar elementos
- Contraste de escala entre título e informações secundárias
- Contraste de valor para garantir legibilidade

### 3. Composição
- Equilíbrio entre elementos visuais e textuais
- Uso eficaz do espaço negativo
- Alinhamento e ritmo visual

### 4. Adaptação Multiplataforma
- Compreensão das especificidades de cada formato
- Manutenção da identidade visual em diferentes dimensões
- Otimização para o contexto de utilização
